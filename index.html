Carly Mahony

I’m Carly, a user experience designer with a passion for digital solutions.

My UX Skills

User Research

One of my strengths is empathising with users and I believe that every deisgn project starts with a thourough understanding of users and the problems they experience. I have used tools such as affinity diagran, user personas, user journey maps and problem statement generation. In the research phase, I often use tools such as Miro, Figma and the Google suite.

User Interface Design

I have completed several design projects using tools such as Adobe XD and Figma. I like to start designing with low-fidelity prototypes and get user feedback early in the design process. I attempt to test prototypes as much as possible before finalizing designs in a high-fidelity prototype.

Web Development

I am code-coversational and have experience using applications such as Visual Studio Code and GitHub I have expereince with html, css, JavaScript, jQuery and Bootstrap.

My Work

Remotely: A travel app

An project from my UX/UI bootcamp at the University of Utah was to design a mobile app related to travel. For this project, I worked with one classmate to design a mobile app for vacationers who are looking to book a vacation that is ‘off the beaten path’.

A Non-profit Website Redesign

As a group of three UX/UI design student, we chose a local non-profit and redesigned their website. This project included a responsive web and mobile redesign for the non-profit, The Christmas Box International.

Fridge Fresh: A mobile app

This project was a challenge to solve an everyday problem. My teammates and I designed an app that would allow users to keep track of what groceries they had in their refrigerator. The app is designed to allow users to scan barcodes and also reccomends recipes bason on the ingredients that you have available.
